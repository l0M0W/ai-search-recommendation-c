# 🚀 AI-based Search & Personalization Recommendation System in C

## 🧑‍💻 개발자 정보 및 진로 계획 (Developer Intro & Career Path)
- **이름:** 홍길동 (학과 및 학번 기재 부문)
- **진로 목표:** 인공지능(AI) 기반 데이터 분석가 및 소프트웨어 엔지니어
- **진로 계획 및 비전:** 현재 빅데이터 분석과 사용자의 행동 패턴을 스스로 학습하는 인공지능 알고리즘에 큰 관심을 가지고 있습니다. 특히 시스템 자원이 제한된 환경에서도 빠르고 가볍게 구동되는 인프라를 구축하기 위해, 로우 레벨 언어인 C언어를 활용하여 알고리즘을 밑바닥부터(From Scratch) 직접 구현하는 역량을 기르고자 합니다. 
  궁극적으로는 사용자가 자신의 관심사를 일일이 기억하거나 반복해서 검색하지 않아도, 과거 검색 데이터와 키워드 가중치를 분석하여 선제적으로 유용한 정보를 제안하는 '지능형 맞춤화(Hyper-Personalization) 서비스'를 개발하는 엔지니어로 성장할 것입니다.

---

## 📌 프로젝트 소개 (Project Overview)
본 프로젝트는 사용자가 매번 번거롭게 관심 키워드를 입력하고 검색하는 수고를 덜어주기 위해 설계된 **"C언어 기반 개인 맞춤형 AI 검색 및 추천 엔진"**입니다. 

### 💡 핵심 기능
1. **즉시 검색 (Instant Search):** 사용자가 원하는 키워드를 입력하면, 데이터베이스에서 매칭되는 정보를 즉각 탐색하여 출력합니다.
2. **관심사 기반 자동 추천 (AI-driven Recommendation):** 사용자가 검색한 기록과 누적된 관심도 지표(Weight)를 분석하여, 사용자가 별도의 검색어를 입력하지 않아도 시스템이 스스로 판단하여 사용자의 취향에 부합하는 추천 콘텐츠를 자동으로 화면에 제시합니다.
